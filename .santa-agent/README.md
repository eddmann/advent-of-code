# santa-agent

Autonomous agent that translates existing Advent of Code solutions (TypeScript, Python, Rust, etc.) into idiomatic [santa-lang](https://eddmann.com/santa-lang/) using the Claude Agent SDK.

## How it works

1. Discovers which years/days are missing santa-lang solutions but have solutions in other languages
2. Reads the source solution and mines test files for example inputs/outputs
3. Fetches the puzzle description and your submitted answers from adventofcode.com
4. Invokes Claude per solution with the santa-lang spec, idiom guide, source code, and puzzle context
5. Claude writes the `.santa` file, runs `santa-cli -t` to test, fixes failures, and verifies answers against AoC

Each solution is tagged with `// Generated by santa-agent from <Language> solution`.

## Usage

```bash
AOC_SESSION=<cookie> bun run src/index.tsx                    # generate all missing solutions
AOC_SESSION=<cookie> bun run src/index.tsx --year 2015        # one year
AOC_SESSION=<cookie> bun run src/index.tsx --year 2015 --day 1  # one puzzle
bun run src/index.tsx --dry-run                               # preview work queue
```

## Requirements

- Bun
- `AOC_SESSION` env var with a valid adventofcode.com session cookie

The `santa-cli` binary (comet) is automatically downloaded on first run.
